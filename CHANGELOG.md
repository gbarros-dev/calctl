# Changelog

## 0.1.0-dev

- Bootstrap `calctl` as a native Swift/EventKit macOS Calendar CLI.
- Add `doctor`, `calendars`, `agenda`, `search`, `add`, `update`, and `delete`.
- Add JSON-first output and machine-friendly error responses.
- Add EventKit permission prompting via `doctor --request-access`.
- Add parser and search-matching tests.
